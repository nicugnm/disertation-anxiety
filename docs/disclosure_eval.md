# User-level self-disclosure evaluation

Each trained model scores every post by the held-out disclosure-test users, the per-post scores are aggregated per user (mean), and the per-user score is compared against the user's self-disclosure label. **with** = the explicit “I was diagnosed…” posts are included; **masked** = those posts are removed first, so the score reflects only the *implicit* signal in the user's other posts (the honest generalization number).
