### Skills to Know About
- [/agents](https://docs.claude.com/en/docs/claude-code/sub-agents) — manage
  subagents (e.g. Explore for read-only research, general-purpose for multi-step
  tasks). Useful when a question would span many files.
- [/model](https://docs.claude.com/en/docs/claude-code/slash-commands) — switch
  between Opus / Sonnet / Haiku. Opus for heavy reasoning; Sonnet for fast
  edits; Haiku for cheap bulk operations.
- [/rate-limit-options](https://docs.claude.com/en/docs/claude-code/slash-commands)
  — manage rate-limit and token usage when working through long sessions.
